# Catatan Keuangan Sederhana

Aplikasi ini adalah website catatan keuangan sederhana untuk mencatat pemasukan, pengeluaran, dan mengelola kategori keuangan.

## Fitur
- Dashboard yang menampilkan ringkasan keuangan.
- Daftar transaksi.
- Daftar kategori.
